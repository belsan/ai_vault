# Projects

Project ideas — things to potentially build, demonstrate, or hand off. The goal is "park ideas so they survive past the moment of having them," per the corpus README.

Conventions: one `.md` per idea. Keep the file name short and stable, even if the working title evolves. When an idea is picked up and worked on, that's a fork into a separate repo or workstream — leave the seed here.

---

## Current ideas

- **`flipbook-style-ai-ui.md`** — Use the `flipbook.page` interaction pattern (image-as-UI, multimodal LLM interprets clicks) as a basis for a fully-AI-generated UI for our product, and as a teaching example covering diffusion + prompt-gen LLM + agent loop + streaming + multimodal LLM in one demo.
- **`ldt_solver_for_hdl.md`** — An LDT-style (Lattice Deduction Transformer) learned formal-verification assistant for SystemVerilog/HDL: a neural deducer proposes invariants, environment assumptions, abstractions, and minimized counterexample traces over RTL abstract domains, while SAT/SMT/PDR stays the source of truth. Seeded by the Lattice Deduction Transformers paper (`resources/models-and-training/2026_lattice-deduction-transformers.pdf`).
