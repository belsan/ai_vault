# Graph Drawing

Reference material for the RTLDraw work — auto-layout of directed graphs / RTL schematics. Originally collected for an email to Samuel; preserved here so the references survive.

---

## Papers (offline)

### Configurable Graph Drawing Algorithms — Pohlmann, 2011 (Master's thesis)
- **URL:** https://www.tcs.uni-luebeck.de/downloads/papers/2011/2011-configurable-graph-drawing-algorithms-jannis-pohlmann.pdf
- **Local:** `2011_pohlmann_configurable-graph-drawing-algorithms.pdf`
- **Summary:** Lübeck master's thesis — many of the canonical examples reproduced in RTLDraw come directly from this. Particularly good as an example reference.

### A Technique for Drawing Directed Graphs — Gansner, North, Vo, 1993 (Graphviz / TSE)
- **URL:** https://www.graphviz.org/documentation/TSE93.pdf
- **Local:** `1993_graphviz_drawing-directed-graphs.pdf`
- **Summary:** The Graphviz layered-layout paper. Foundational reading for anyone implementing layered drawings.

### KLay Layered: Integral Layered Graph Drawing — 2013 (Kiel / JVLC)
- **URL:** https://rtsys.informatik.uni-kiel.de/~biblio/downloads/papers/jvlc13.pdf
- **Local:** `2013_klay-layered-integral-layered-graph-drawing.pdf`
- **Summary:** Describes a layered-drawing algorithm that, per Sandro's note, may match the exact problem setting we hit with RTLDraw. Marked as worth a deeper dive.
